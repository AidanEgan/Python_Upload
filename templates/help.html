Want to request a feature? The PiBrary is constantly being updated. See below for how you can update your software

HOW TO UPDATE YOUR PIBRARY

  1. Set up SSH or simply connect the Rasberry Pi to a monitor

  2. Navigate to the directory containing the code for the Pibrary

  3. Open up a command prompt and point it to this directory using the "cd" command

  4. Run the update.bat script by simply typing "update.bat" into the command prompt
